What I'm Up To

W hile I'm often entertaining the idea of more projects than I can feasibly complete I currently have two that I am trying to focus in on.

100 Days of Github

This isn't really a project so much as a motivational tool. The idea is that for 100 days I will push something to github every day. It doesn't matter what it is, it could be a README edit it just has to be something. By doing this I'm able to keep myself thinking about what I want to accomplish constantly. Additionally I can find myself reading all day and ending up with nothing to show for it at the end. This forces me to make some tangible, or at least recorded, progress on my ideas every day.

Rocket-Calc

Reading the text How to Design, Build and Test Small Liquid-fuel Rocket Engines got me incredibly excited about bi-propellant rocket engines. It offers a design concept for one dumbed down enough for an amature to design and put together themselves! After going through the example design calculations I considered the educational and practical value in making a more robust command line tool that could perform these same calculations. The end goal is to have support for many materials, propellants, oxidizers, coolants but development has only just begun. The full source code and documentation can be found on my github.
